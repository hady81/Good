# لعبة شد الحبل (Tug of War) 🏆

لعبة متعددة اللاعبين تعتمد على المنافسة في سحب الحبل بين فريقين.

## ✨ الميزات المخطط لها
- وضع لاعب ضد لاعب (PvP).  
- نظام نقاط وتحديات.  
- دعم للغة العربية.  

## 🛠 التقنيات المستخدمة
- **الواجهة الأمامية:** HTML, CSS, JavaScript.  
- **الخلفية:** Firebase (للمزامنة بين اللاعبين).
